💿Instalacja

macOS

macOS 12 Monterey+

Pobierz plik .dmg, otwórz go i przeciągnij ikonę WavesConverter do folderu Applications. Przy pierwszym uruchomieniu kliknij prawym przyciskiem myszy ikonę → Otwórz → ponownie Otwórz w oknie Gatekeepera.

Windows

Windows 10+ (64-bit)

Pobierz plik .exe i uruchom instalator. Jeśli pojawi się niebieskie okno SmartScreen, kliknij Więcej informacji → Uruchom mimo to. Instalacja zajmuje kilka sekund.

🍎
Wskazówka macOS: Jeśli Gatekeeper blokuje aplikację, przejdź do Preferencji systemowych → Prywatność i bezpieczeństwo → kliknij „Otwórz mimo to". Wystarczy zrobić to raz.
🛡️
Dlaczego SmartScreen? WavesConverter jest bezpieczny i open source, ale nie jest jeszcze podpisany płatnym certyfikatem Microsoft. To normalne dla nowego oprogramowania indie. Kod źródłowy jest publiczny na GitHub.

🎬Pobieranie filmów

Aby pobrać pojedynczy film z YouTube, wykonaj poniższe kroki:

1

Wklej URL — skopiuj link do filmu (youtube.com/watch, youtu.be lub Shorts) i naciśnij ⌘V (macOS) lub Ctrl+V (Windows). Link automatycznie pojawi się w polu i zacznie się pobieranie metadanych.

2

Kliknij Fetch — aplikacja pobierze tytuł, miniaturę i dostępne formaty z YouTube.

3

Wybierz format i jakość — przełącz między Video / Audio, wybierz format (MP4, MKV, MOV, WebM dla wideo; MP3, WAV, FLAC, M4A dla audio) oraz rozdzielczość lub bitrate.

4

Dodaj do kolejki — kliknij Add to Queue lub naciśnij ⌘D. Film trafi do zakładki Queue.

5

Start — przejdź do zakładki Queue i kliknij Start All lub Spacja. Obserwuj postęp i ETA.

⌨️
Skrót ⌘V: Wystarczy skopiować link do schowka — po kliknięciu okna aplikacji link zostanie automatycznie wklejony i natychmiast zostanie wywołany Fetch. Nie musisz ręcznie klikać w pole URL.
🖼️
Pobieranie miniaturki HD: Po wczytaniu metadanych filmu, obok podglądu miniatury pojawi się dedykowany przycisk „Miniaturka HD”. Kliknięcie go pobierze bezpośrednio plik graficzny miniatury w najwyższej dostępnej rozdzielczości bezpośrednio do Twojego folderu pobierania.

📋Pobieranie playlist

WavesConverter obsługuje pełne playlisty YouTube — możesz pobrać wszystkie filmy lub wybrać konkretne pozycje.

1

Wklej URL playlisty — link musi zawierać parametr list= (np. youtube.com/playlist?list=PL…). Naciśnij Fetch.

2

Wybierz pozycje — aplikacja wyświetli listę wszystkich filmów. Zaznacz lub odznacz poszczególne filmy. Przyciski All / None zaznaczają lub odznaczają wszystkie.

3

Ustaw globalne ustawienia — format, jakość i folder docelowy w sekcji Global settings zostaną zastosowane do wszystkich zaznaczonych filmów.

4

Nadpisz pojedyncze filmy — kliknij ikonę ustawień przy konkretnym filmie, aby zmienić format lub jakość tylko dla niego. Ustawienie per-item ma pierwszeństwo nad globalnym.

5

Add Selected to Queue — kliknij przycisk, aby dodać wszystkie zaznaczone filmy do kolejki.

📦Batch paste

Batch paste pozwala wkleić wiele linków YouTube naraz — każdy w osobnej linii. Aplikacja wykrywa wiele URLi i automatycznie przełącza się w tryb batch.

1

Skopiuj wiele URLi — każdy link w osobnej linii (oddzielone znakiem nowej linii). Wklej do pola URL (⌘V).

2

Panel Batch pojawi się automatycznie — zobaczysz listę wykrytych URLi z nagłówkiem informującym o ich liczbie. Tytuły zostaną pobrane dopiero podczas pobierania.

3

Ustaw format dla wszystkich — w sekcji Batch settings wybierz typ (Video / Audio), format, jakość i folder wyjściowy. Te ustawienia zostaną zastosowane do całego batcha.

4

Add All to Queue — kliknij przycisk, aby dodać wszystkie URL do kolejki jednocześnie.

💡
Batch paste sprawdza się idealnie gdy masz listę ulubionych filmów lub chcesz pobrać kilka konkretnych wideo z różnych kanałów naraz — bez potrzeby fetchowania każdego z osobna.

🔄Konwerter plików

Zakładka Convert pozwala konwertować dowolny lokalny plik multimedialny na inny format. Wszystko odbywa się lokalnie przy użyciu ffmpeg — żadnego uploadu.

1

Wybierz plik — przeciągnij plik na obszar drop zone lub kliknij Browse i wybierz plik z dysku. Obsługiwane wejście: MP4, MKV, MOV, AVI, WebM, MP3, WAV, FLAC, M4A, AAC, OGG i inne.

2

Wybierz format wyjściowy — z listy dostępnych formatów. Dla wideo możesz też zmienić rozdzielczość i bitrate audio.

3

Konwertuj — kliknij przycisk Convert. Postęp jest widoczny w czasie rzeczywistym. Przekonwertowany plik zostanie zapisany w tym samym folderze co oryginał (lub w wybranym folderze wyjściowym).

4

Tworzenie animowanych GIF-ów — po wybraniu formatu wyjściowego GIF, pojawią się dodatkowe opcje umożliwiające określenie czasu rozpoczęcia (Start Time) oraz czasu trwania (Duration). WavesConverter użyje profesjonalnego algorytmu podwójnego przejścia FFmpeg dla zachowania najwyższej jakości palety barw i płynności.

⚡Kolejka i ETA

Zakładka Queue to centrum operacyjne WavesConverter — tu zarządzasz wszystkimi pobieraniami.

Równoległe pobierania

Aplikacja może pobierać kilka plików jednocześnie. Limit równoległych pobierań ustawiasz w Ustawieniach → Concurrent Downloads (domyślnie: 3). Każde pobieranie ma własny pasek postępu, ETA i aktualną prędkość.

Sterowanie kolejką

▶

Start All — uruchamia wszystkie oczekujące pobierania (do limitu współbieżności). Skrót: Spacja.

⏸

Pause / Resume — zatrzymuje i wznawia pobieranie. Częściowo pobrane dane są zachowane.

↺

Retry — ponawia pobieranie które zakończyło się błędem. Kliknij ikonę przy pozycji lub użyj przycisku Retry Failed.

✕

Usuń — usuwa pozycję z kolejki. Nie usuwa pobranego pliku z dysku.

⏱️
ETA (szacowany czas do końca) i prędkość pobierania aktualizują się na żywo. Przy wolnym połączeniu ETA może być niedokładne na początku — stabilizuje się po kilku sekundach.

📖Historia

Zakładka History zapisuje wszystkie ukończone pobierania — tytuł, format, rozdzielczość, datę i ścieżkę pliku.

Otwieranie pliku z historii

Kliknij ikonę folderu przy dowolnej pozycji, aby otworzyć folder zawierający pobrany plik w Finderze (macOS) lub Eksploratorze plików (Windows). Możesz też kliknąć tytuł, aby otworzyć plik bezpośrednio.

Czyszczenie historii

Przycisk Clear History usuwa całą historię z aplikacji. Pliki na dysku pozostają nienaruszone.

Panel statystyk (Dashboard)

W górnej części zakładki History znajduje się zaawansowany panel statystyk, który pokazuje ogólne podsumowanie Twojej aktywności: łączną liczbę pobranych plików, ich łączny rozmiar na dysku oraz szacowany czas zaoszczędzony dzięki aplikacji.

⚙️Ustawienia

Zakładka Settings pozwala dostosować zachowanie aplikacji do własnych potrzeb.

Folder domyślny

Ustaw domyślny folder pobierania — każde nowe pobieranie trafi tam automatycznie, chyba że zmienisz folder ręcznie przy konkretnym pobraniu.

Limit równoległych pobierań

Określa ile plików może być pobieranych jednocześnie (1–10). Wyższa wartość = szybciej przy szybkim łączu, ale też większe zużycie zasobów. Domyślnie: 3.

Domyślny format i jakość

Ustaw domyślny typ (Video/Audio), format (MP4, MP3 itp.) i jakość. Możesz je nadpisywać przy każdym pobieraniu osobno.

Szablon nazwy pliku

Wybierz schemat nazewnictwa: %(title)s (samo tytuł), %(uploader)s - %(title)s (autor i tytuł), %(title)s [%(id)s] (z ID) lub wpisz własny wzorzec kompatybilny z yt-dlp.
