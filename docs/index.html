Why WavesConverter

Everything you need.
Nothing you don't.

Built for speed, privacy, and control. No subscriptions. No cloud. No tracking.

📋

Playlist Downloads

Download entire playlists or cherry-pick videos. Set global settings or override per-item format and quality.

⚡

Download Queue

Queue multiple downloads and run them concurrently. Pause, resume, retry failed items — full control.

🎛️

Full Quality Control

Choose resolution from 360p to 4K, audio bitrate from 96 to 320 kbps. Apply globally or per video.

🔄

Local File Converter

Convert any local media file to any format. Change resolution, bitrate, container — all 100% offline.

🔒

100% Offline

No servers, no accounts, no data collection. Everything runs locally using yt-dlp and ffmpeg.

✨

Auto Updates

Silent background updates via GitHub Releases. A pill appears when a new version is ready to install.

📁

Custom Output

Choose your download folder and filename template. Supports %(title)s, %(uploader)s, %(id)s and more.

🎨

Premium Design

Dark purple glassmorphism UI with animated accents. Frameless window with native macOS & Windows feel.
