✦ Funkcje

Wszystko czego potrzebujesz.
Nic zbędnego.

Zbudowane z myślą o szybkości, prywatności i pełnej kontroli. Żadnych subskrypcji. Żadnej chmury. Żadnego śledzenia.

📋

Obsługa playlist

Pobieraj całe playlisty lub wybieraj poszczególne filmy. Ustaw globalne ustawienia lub nadpisuj format i jakość dla każdego filmu osobno.

⚡

Równoległa kolejka

Dodaj wiele pobrań do kolejki i uruchom je jednocześnie. Pauzuj, wznawiaj lub ponawiaj nieudane — pełna kontrola jednym kliknięciem.

🎛️

Pełna kontrola jakości

Wybieraj rozdzielczość od 360p do 4K i audio od 96 do 320 kbps. Ustawiaj globalnie lub osobno dla każdego filmu.

🔄

Konwerter plików lokalnych

Konwertuj dowolny lokalny plik multimedialny na dowolny format. Zmień kontener, rozdzielczość, bitrate — 100% offline z ffmpeg.

📖

Historia pobrań

Każde zakończone pobranie zapisane ze ścieżką pliku, formatem i datą. Otwórz dowolny plik bezpośrednio z historii.

⌨️

Skróty klawiszowe

⌘V auto-wklej URL, ⌘D dodaj do kolejki, ⌘1–5 przełączaj zakładki, Spacja start/pauza — szybkość w DNA.

🔔

Powiadomienia systemowe

Natywne powiadomienie pojawia się gdy pobranie się kończy — nawet jeśli aplikacja jest w tle.

🔒

100% Offline

Żadnych serwerów, kont, analityki. Wszystko działa lokalnie dzięki yt-dlp i ffmpeg dołączonym do aplikacji.

✨

Automatyczne aktualizacje

Ciche aktualizacje przez GitHub Releases. Baner pojawia się gdy nowa wersja jest gotowa — jedno kliknięcie do instalacji.

✦ Jak działa

Trzy kroki.
I tyle.

Żadnych kont, rozszerzeń do przeglądarki ani komplikacji.

1

Wklej link

Skopiuj dowolny link z YouTube — pojedynczy film, playlistę lub Shorts. Naciśnij ⌘V i link automatycznie wkleja się do aplikacji. Obsługuje youtube.com, youtu.be, shorts i więcej.

2

Wybierz format i jakość

Wybierz Video lub Audio, potem format (MP4, MP3, WAV…) i jakość (1080p, 4K, 320kbps…). Dodaj do kolejki — możesz dodać ile chcesz naraz.

3

Start — pliki lądują w folderze

Obserwuj paski postępu. Dostaniesz powiadomienie gdy skończy. Pliki znajdziesz dokładnie tam gdzie ustawiłeś. To cały przepływ.

⚠️ Windows może pokazać ostrzeżenie SmartScreen — oto jak je ominąć ▼

WavesConverter jest bezpieczny i open source, ale ponieważ nie jest jeszcze podpisany płatnym certyfikatem Microsoft, Windows Defender SmartScreen może oznaczyć go jako nieznaną aplikację. Jest to normalne dla nowego oprogramowania indie. Wykonaj poniższe kroki, żeby zainstalować:

1
Pobierz plik .exe i kliknij go dwukrotnie, żeby uruchomić instalator.
2
Pojawi się niebieskie okno SmartScreen z napisem „Ten komputer jest chroniony przez system Windows". Nie zamykaj go.
3
Kliknij „Więcej informacji" (mały link pod głównym tekstem). Okno się rozszerzy.
4
Kliknij przycisk „Uruchom mimo to", który pojawi się na dole. Instalacja rozpocznie się normalnie.
Możesz w każdej chwili zweryfikować kod źródłowy na GitHub — wszystko jest jawne i audytowalne.
